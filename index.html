COVID-19, Argentina

web en construcción
En esta página web se presenta información de la evolución del COVID-19 en Argentina. Se proporciona información sobre el número de infecciones y muertes por día, basada en datos del Ministerio de Salud de Argentina. Además, se presentan pronósticos básicos de las variables mencionadas. Los mismos se calcularon en base a ajustes de funciones exponenciales a partir de datos de los últimos 7 y 14 días. El código fuente para la generación de este sitio web es de open-source y puede descargarse desde el siguiente enlace.
